# PromptC Pattern Library

This file collects the first three flagship PromptC patterns for launch and dogfooding.

## 1. GitHub Issue -> Implementation Brief

**Input intent**

> Turn this GitHub issue into a safe implementation brief with risks, constraints, and a checklist.

## 2. PR Description -> Review Brief

**Input intent**

> Compile this PR description into a focused review brief with policy checks and likely regression areas.

## 3. Research Task -> Safe Agent Brief

**Input intent**

> Turn this research request into a safe brief for an agent, including constraints, sources, and execution limits.
